# Project Notes

This project was developed as part of my Tableau analytics learning path during the ACA Data Analytics bootcamp.

The goal was to transform retail transaction data into business-focused dashboards that support analytical decision-making.

## Project Scope

The repository includes two Tableau dashboards:

1. Sales Analysis Dashboard
2. Customer Analysis Dashboard

Both dashboards are based on the same retail dataset and are connected by a common business theme: understanding retail performance from both sales and customer perspectives.

## Sales Analysis Scope

The sales dashboard focuses on business performance.

Main analysis areas:

- KPI overview
- Sales trends
- Geographic sales analysis
- Product performance
- Sales by weekday and hour
- Period-over-period comparison

## Customer Analysis Scope

The customer dashboard focuses on customer behavior and customer value.

Main analysis areas:

- Customer KPI overview
- Customer growth
- Customer segmentation
- Revenue per customer
- Purchase frequency
- RFM analysis
- Cohort retention analysis
