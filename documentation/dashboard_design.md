# Dashboard Design Notes

This project follows a clean, structured, executive-style dashboard design approach.

The goal of the dashboards is to make retail performance and customer behavior easy to understand through clear KPI sections, analytical charts, and interactive Tableau layouts.

## Design Goals

- Clear visual hierarchy
- KPI-first analytical structure
- Minimal and consistent formatting
- Business-focused storytelling
- Interactive exploration
- Presentation-ready layout

## Dashboard Layout

The dashboards were designed with fixed desktop layouts to maintain consistent spacing, alignment, and presentation quality.

- Sales Analysis Dashboard: fixed desktop layout
- Customer Analysis Dashboard: extended vertical dashboard layout

The customer dashboard uses a taller vertical layout because it includes more analytical sections such as customer KPIs, segmentation, frequency analysis, RFM analysis, and cohort retention.

## Visual Hierarchy

The dashboards are organized from high-level overview to deeper analysis.

Recommended flow:

1. KPI section
2. Trend analysis
3. Geographic, product, or segmentation analysis
4. Customer behavior analysis
5. Retention and advanced analytics

The KPI section appears first because it gives users an immediate understanding of business performance before they explore detailed charts.

## Design Principles

- Keep KPI cards visually dominant
- Use consistent colors and fonts
- Avoid excessive labels and borders
- Align containers carefully
- Use spacing to separate dashboard sections
- Use color to communicate meaning, not decoration
- Keep charts readable and business-oriented

## Tableau Layout Practices

The dashboards use Tableau containers to keep the layout structured and easier to maintain.

Main layout practices:

- Vertical containers for main dashboard structure
- Horizontal containers for KPI cards
- Consistent padding between dashboard sections
- Fixed layout sizing for presentation consistency
- Minimal borders and clean formatting

## Dashboard Interaction

The dashboards are designed to support interactive analysis through filters, parameters, tooltips, and dashboard actions.

Interaction goals:

- Allow users to explore different time periods
- Help users compare current and previous performance
- Support customer and sales segmentation
- Make dashboard exploration simple and intuitive

## Design Notes

The visual style is intentionally clean and minimal. The dashboards are designed to feel analytical, professional, and presentation-ready rather than decorative or overcrowded.
